# EasyEncryption
Simple encryption tool using XOR

## Caution
- Because of the C++ version, ShiftJIS is used, so if you open the output file with UTF-8, etc., you will get inaccurate results.

## Requirement
C++11 or above.

gcc 4.8 or above.\
or\
MSVC 14.0 or above.

## Build Instructions
Build using g++.\
`g++ main.cpp -o OPTIONAL_NAME MORE_OPTIONS`

the end.
